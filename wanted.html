Open Research Topics

Interesados contactarme al email juanpablo.sandoval (at) ing.puc.cl

Topic Goal Requirement Funding Oportunities
Test Quality Enhancer Analizar y mejorar la calidad de las pruebas de unidad (unit-testing) de forma automatica. Estudiante de ultimos semestres que quieran realizar su tesis te magister (y a la vez obtener el doble título académico de ingeniero UC). Este proyecto cuenta con financiamiento que permite pagar al tesista interesado.
Test Visualization Tool Crear una herramienta visual que permita a los desarrolladores analizar el coverage y la diversidad de las pruebas de unidad. Estudiante de ultimos semestres que quieran realizar su tesis te magister (y a la vez obtener el doble título académico de ingeniero UC). Este proyecto cuenta con financiamiento que permite pagar al tesista interesado.
Pharo Development Tool Mejorar una de las herramientas de desarrollo de software de Pharo. Las posibles herramientas a mejorar son debugger, versionamiento, linter, testing tools, code completion, performance profilers, markdown, virtual machine, entre otras. Abierto para estudiantes de pregrado o postgrado. Estos proyectos se realizan en colaboracion con INRIA France. Dependiendo al rendimiento y al proyecto es posible obtener financiamiento para realizar un visita de investigacion a INRIA-Lille, Francia.
